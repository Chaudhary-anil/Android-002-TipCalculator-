# Android-002-TipCalculator App

A simple Tip Calculator built with **Jetpack Compose** and **Kotlin** for android, helps to calculate tip amount, to calculate total and split the total among multiple people.

## Features

- Input **bill amount**
- Adjust **tip percentage**
- Adjust **number of people**
- Calculate **total bill** and **per person** amount
- Reset all values using **Reset** btn
- Displays amount in **Nepali Rupees (Rs)**
- Built with **Jetpack Compose**
